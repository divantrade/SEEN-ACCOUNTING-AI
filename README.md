# 🎬 SEEN Accounting AI

نظام محاسبة ذكي متكامل مبني على Google Sheets مع بوت Telegram تقليدي وبوت ذكاء اصطناعي،
مصمم خصيصاً لإدارة حسابات الإنتاج الإعلامي لشركة **Start Scene Media Produksiyon Limited**.

## ✨ المميزات

### المحاسبة والحركات المالية
- تسجيل الحركات المالية (مدين / دائن / تسوية) مع دعم متعدد العملات (USD, TRY, EGP)
- إدارة الموردين والعملاء والممولين
- تتبع المشاريع والبنود والوحدات الإنتاجية
- نظام الاستحقاقات والتنبيهات للمدفوعات القادمة
- تسوية بنكية ومتابعة النقدية
- كشف الحركات المكررة تلقائياً (بصمة: طبيعة + مبلغ + طرف + عملة)

### التقارير
- كشوفات حساب PDF احترافية (مورد / عميل / ممول)
- تقارير ربحية المشاريع وتكلفة الوحدات
- القوائم المالية (قائمة الدخل، الميزانية العمومية، ميزان المراجعة)
- دفتر الأستاذ العام وقيود اليومية
- تقارير المصروفات والإيرادات والتدفق النقدي
- تقرير تحليل المصروفات الديناميكي (عمومية / مباشرة-أفلام / مباشرة بدون مشروع)
- محرك تقارير موحد `buildSummaryReport_` يقلل التكرار

### بوت Telegram الذكي
- بوت ذكاء اصطناعي واحد يعمل بـ Gemini Function Calling
- تسجيل حركات بالمحادثة الطبيعية بالعربية
- بحث ذكي بالعربية مع دعم الأخطاء الإملائية (Fuzzy Search 5 مستويات)
- نظام مراجعة واعتماد الحركات مع إشعارات فورية
- 12 تقرير PDF متاح مباشرة من البوت
- تعلم تفضيلات المستخدم من آخر حركاته
- أسعار صرف ديناميكية من آخر الحركات الفعلية
- إرفاق مستندات وصور مع الحركات

## 🚀 التثبيت والإعداد

### 1. استنساخ المشروع
```bash
git clone https://github.com/divantrade/SEEN-ACCOUNTING-AI.git
cd SEEN-ACCOUNTING-AI
```

### 2. تثبيت clasp
```bash
npm install -g @google/clasp
```

### 3. تسجيل الدخول
```bash
clasp login
```

### 4. ربط المشروع بـ Google Apps Script
ملف `.clasp.json` يحتوي على `scriptId` المرتبط بالمشروع. إذا كنت تبدأ من جديد:
```bash
clasp clone <SCRIPT_ID>
```

### 5. رفع الكود
```bash
clasp push
```

### 6. إعداد Telegram Bot
1. أنشئ بوت جديد عبر [@BotFather](https://t.me/BotFather)
2. احفظ الـ Token في Script Properties:
   - `AI_BOT_TOKEN` - توكن البوت الذكي
3. فعّل Time-driven Trigger لـ `processAIBotUpdates()` (كل دقيقة)

## 📖 الاستخدام

### من Google Sheets (القائمة الرئيسية)
بعد فتح الجدول، تظهر قائمة **💼 نظام المحاسبة** وتحتوي على:
- **العمليات اليومية**: إضافة حركة، أوردر مشترك، فاتورة قناة
- **التقارير التشغيلية**: استحقاقات، مستحقات بالبنود، كشف حساب فرعي
- **كشوفات الحساب**: مورد / عميل / ممول (PDF)
- **تقارير المشاريع**: ربحية، تكلفة وحدات، عمولات
- **التقارير الإجمالية**: ملخص موردين، ممولين، مصروفات، إيرادات، تدفق نقدي
- **القوائم المالية**: قائمة الدخل، الميزانية، ميزان المراجعة، دفتر الأستاذ
- **الموازنات**: إضافة ومقارنة
- **البنك والنقدية**: تحديث وتسوية

### من Telegram Bot (الذكي)

يمكنك التحدث بشكل طبيعي بالعربية:
> "اتفقت مع أحمد نايل على رسم فيلم بل كلنتون بقيمة 400 دولار"

> "دفعت 500 دولار لأحمد مونتاج نقدي"

> "استلمت 5000 دولار من قناة الجزيرة"

أو استخدم الأوامر:
```
/start      - بدء المحادثة
/reports    - قائمة التقارير (12 تقرير PDF)
/مساعدة    - المساعدة
/الغاء     - إلغاء العملية الحالية
```

## 📝 سجل التعديلات

### الفرع: `claude/analyze-test-coverage-Pjqg5`

#### 1. إزالة التكرار وتوحيد الكود
- **توحيد الدوال المكررة**: `normalizeArabicText`، `smartArabicMatch`، `searchPartiesByName`، `deleteAIBotWebhook` كانت مكررة بين عدة ملفات. تم توحيدها في مكان واحد (`FuzzySearch.js` و `AIConfig.js`)
- **توحيد 23 timezone مكرر**: استبدال كل `'Asia/Istanbul'` المكتوبة يدوياً بـ `CONFIG.COMPANY.TIMEZONE`
- **إعادة تسمية `getOrCreateMonthFolder`**: إلى `getOrCreateAttachmentsMonthFolder_` لتجنب تعارض الأسماء مع `PDFGenerator.js`
- **توحيد إنشاء كشوفات الحساب**: `generateStatementForBot_` و `generateUnifiedStatement_` تم دمجهما في `renderStatementSheet_` بخيارات قابلة للتخصيص

#### 2. محرك التقارير الموحد
- **إنشاء `buildSummaryReport_`**: محرك واحد يتولى القراءة والكتابة والتنسيق لـ 5 تقارير ملخصة بدلاً من تكرار ~200 سطر في كل تقرير
- التقارير الموحدة: المشروعات التفصيلي، الموردين، المصروفات، الإيرادات، الممولين

#### 3. تقرير تحليل المصروفات الجديد
- **`generateDynamicExpenseReport()`**: تقرير ديناميكي يصنف المصروفات إلى:
  - عمومية (overhead)
  - مباشرة مرتبطة بأفلام
  - مباشرة بدون مشروع
- لوحة ملخص (سطور 1-7) + جدول موحد مع فلتر (سطر 8+)
- متاح من القائمة في الشيت ومن البوت كـ PDF

#### 4. توسيع تقارير البوت (7 → 12 تقرير)
- إضافة 5 تقارير جديدة للبوت: التدفقات النقدية، قائمة الدخل، المركز المالي، الممولين، المشروعات
- إنشاء `handleGenerateDirectReport_()` كدالة موحدة لتجنب تكرار handlers

#### 5. حذف البوت التقليدي
- **حذف `TelegramBot.js`** (~3,154 سطر): كان يحتوي على بوت تقليدي بقوائم ثابتة، لكن إدخال الحركات فيه لم يكن مكتملاً
- **نقل 4 دوال مشتركة** إلى `AIBot.js`: `notifyUserApproval`، `notifyUserRejection`، `getFileInfo`، `downloadTelegramFile`
- الدوال المنقولة تستخدم الآن `getAIBotToken()` بدلاً من `getBotToken()` القديم

#### 6. تحسينات الأداء والذكاء
| التحسين | التأثير |
|---|---|
| Long Polling timeout 3s → 25s | تقليل 90% من طلبات API لتليجرام |
| تفعيل cache الجلسات في الذاكرة | أسرع 5x في قراءة/كتابة الجلسات |
| أسعار صرف ديناميكية | يقرأ آخر سعر فعلي من الحركات بدل قيم ثابتة |
| تجميع أدوات Gemini (batch) | طلب واحد لـ Gemini بدل 3-5 متتابعة |
| تعلم تفضيلات المستخدم | يقترح الأطراف والعملات المتكررة من آخر 5 حركات |
| كشف الحركات المكررة | بصمة (طبيعة+مبلغ+طرف+عملة) خلال 10 دقائق |

## 🚦 دليل نشر التعديلات

### الترتيب الآمن لنقل التعديلات إلى Apps Script

**المرحلة 1 - الأساسيات (لا تكسر شيء):**
1. `Config.js` - إعدادات جديدة
2. `FuzzySearch.js` - توحيد دوال البحث
3. اختبر: افتح الشيت، جرّب البحث عن طرف

**المرحلة 2 - التقارير:**
4. `Main.js` - محرك التقارير الموحد + تقرير تحليل المصروفات
5. اختبر: شغّل كل تقرير من القائمة

**المرحلة 3 - البوت والتقارير عبره:**
6. `PDFGenerator.js` - تسجيل التقارير الجديدة
7. `ReportsConfig.js` - أزرار التقارير
8. `ReportsBot.js` - معالجات التقارير
9. اختبر: أرسل `/reports` للبوت

**المرحلة 4 - تحسينات الأداء:**
10. `SmartAgent.js` - أسعار صرف + تفضيلات + تجميع أدوات
11. `BotSheets.js` - كشف التكرار
12. `AIBot.js` - Long Polling + cache الجلسات
13. اختبر: أرسل حركة عبر البوت وتأكد من السرعة

**المرحلة 5 - حذف البوت التقليدي (الأخيرة):**
14. تأكد أن البوت الذكي يعمل بالكامل
15. احذف trigger الـ `processPendingUpdates` من Apps Script (Triggers → حذف)
16. احذف `TelegramBot.js`
17. اختبر: اعتمد/ارفض حركة وتأكد وصول الإشعار
